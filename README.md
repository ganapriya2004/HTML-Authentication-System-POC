# Bootstrap Authentication UI Project

## Introduction

This project is a simple authentication user interface built using HTML and Bootstrap 5. It is an improved version of a basic HTML-only assignment, where styling and responsiveness have been added using Bootstrap and custom CSS.

The application demonstrates how different authentication pages are connected using navigation links. It does not include any backend functionality and is only focused on front-end design.

---

## Objective

The main purpose of this project is to:

- Understand Bootstrap components
- Build responsive web pages
- Improve UI design using CSS
- Create a structured multi-page application

---

## Technologies Used

- HTML5  
- Bootstrap 5 (CDN)  
- Bootstrap JavaScript Bundle  
- Custom CSS  
- Google Fonts

## Pages Overview

### 1. Login Page (index.html)
- Uses Bootstrap card layout  
- Contains email and password fields  
- Provides links to Register and Forgot Password pages  
- Login button redirects to dashboard  

---

### 2. Register Page (register.html)
- Form for new user registration  
- Includes name, email, password fields  
- Styled using Bootstrap form classes  
- Redirects to login page after registration  

---

### 3. Forgot Password Page (forgot-password.html)
- Allows user to enter email  
- Clean and simple design  
- Redirects to reset password page  

---

### 4. Reset Password Page (reset-password.html)
- Fields for new password and confirmation  
- Styled with Bootstrap  
- Redirects back to login page  

---

### 5. Dashboard Page (dashboard.html)
- Includes a Bootstrap navbar  
- Displays a welcome message  
- Logout button redirects to login page  

---
